# Portfolio

Personal portfolio website for Yuvneet22.

## Overview

This repository contains a single-file static portfolio site:

- portfolio.html — main page showcasing projects, skills, and contact details.

## View locally

Open `portfolio.html` directly in your browser, or run a minimal HTTP server from the repo root:

```bash
# Serve files on http://localhost:8000
python -m http.server 8000
```

Then open: http://localhost:8000/portfolio.html

## Develop

- Edit `portfolio.html` to update content, CSS, or add sections.
- No build tools or dependencies are required — plain HTML/CSS/JS.

## Deploy

- Deploy to GitHub Pages by enabling Pages for the `main` branch, or upload `portfolio.html` to any static host (Netlify, Vercel, S3, etc.).

## Contributing

Open issues or create pull requests to suggest changes. For content updates, edit `portfolio.html` and submit a PR.
